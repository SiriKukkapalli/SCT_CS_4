# Task 4 - Basic Typing Logger Project

## Description
This project is a simple Typing Logger web application developed using HTML, CSS, and JavaScript. It captures user input typed inside a text area and displays it in real time on the screen.

## Features
- Capture text input inside webpage
- Display typed text in real time
- Save typed data as a text file
- Clear logged data
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project files
2. Open `index.html` in any web browser
3. Start typing in the text area
4. Click Save Log to download the text file
5. Click Clear to reset the input

## Project Output
- Displays typed text in real time
- Saves input as `typing_log.txt`
